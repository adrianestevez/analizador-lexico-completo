# Analizador Lexico

Esta actividad consiste en generar un analizador léxico completo con los respectivos símbolos:

*Identificadores= letra (letra|digito)*
*Entero= digito +
*Real= entero.entero
*Operador de adición: "+" | "-"
*Operador de multiplicación: * | /
*Operador de asignación: =
*Operador relacional: < | > | <= | >= | != | ==
*Operador And: &&
*Operador Or: ||
*Operador Not: !
*Parentesis: ( , )
*Llave: { , }
*Punto y coma: ;

Palabras reservadas:

*if
*while
*return
*else
*int
*float
